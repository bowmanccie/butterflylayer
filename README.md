## Local Dev Notes

The website uses a simple HTML + JS include system:
- `site/partials/` holds shared header/footer
- `site/assets/js/loader.js` injects them into pages
- CSS variables live in `site/assets/css/variables.css`

This ensures easy maintenance and zero duplication.
